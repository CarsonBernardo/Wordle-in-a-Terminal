# Wordle-in-a-Terminal
A simple Wordle game written in Python.
## How to play:
1. Make sure _**Wordle.py**_, _**guesses.txt**_ and _**words.txt**_ are all at the same directory.
2. Run **_Wordle.py_**.


## Project specifics:
- 2291 words to be selected from as the mystery word.
- 10757 words marked as valid guesses.
